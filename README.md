# 🎙️ Python Voice Assistant

A Python-based **voice-controlled personal assistant** with speech recognition, Wikipedia integration, battery status check, and more.  
It allows you to interact **hands-free** to perform various daily tasks.

---

## Features
- **Open websites & applications** — launch your favorite sites and programs by voice.
- **Tell time & date** — quickly get the current time and date.
- **Retrieve Wikipedia summaries** — get concise answers from Wikipedia.
- **Check battery status** — know your device’s power level instantly.
- **Take notes** — save notes by speaking.
- **Send emails** — send messages without typing.
- **Tell jokes** — lighten up your day with random jokes.

---

## Technologies Used
- **Python 3**
- **SpeechRecognition** — converts speech to text.
- **Pyttsx3** — text-to-speech engine for voice responses.
- **Wikipedia API** — fetches summaries.
- **psutil** — for battery status.
- **smtplib** — for sending emails.
